# TEAMSPIRIT：中文介面文案書面化（香港繁體）設計稿

日期：2026-05-14  
範圍：`artifacts/teamspirit/src/lib/i18n.tsx`（僅 `translations.zh`）

## 背景／問題

- 現時中文介面存在較多口語化字詞（例如「唔／嘅／揀／仲未／點一下／搞手」等），與產品定位及閱讀體驗不一致。

## 目標

1. 將中文介面文案統一為香港繁體書面語。
2. 稱呼維持使用「你」，不改為「您」。
3. 僅調整系統介面文案來源，避免影響用戶內容（例如留言、評價、用戶輸入）。

## 非目標

- 不改動英文介面文案。
- 不改動示例資料（mock data）及任何用戶生成內容。
- 不變更 i18n key 結構及程式邏輯（避免造成缺字或 key mismatch）。

## 修改範圍與來源

- 只修改：[i18n.tsx](file:///workspace/artifacts/teamspirit/src/lib/i18n.tsx) 內 `translations.zh` 的字串值。
- 不修改其他檔案之中文句子，即使其存在口語字眼（例如 `mockData.ts` 的評語），因其屬示例或內容資料。

## 文案規範（香港繁體書面語）

### 用詞規則

- 避免口語：不使用「唔／咩／嘅／俾／揀／仲未／點一下／開波／夾隊／搞手／波友」等。
- 書面替代（按語境選用）：
  - 唔 → 不／未／無法／不得
  - 嘅 → 的
  - 揀 → 選擇
  - 仲未 → 尚未
  - 俾 → 予以／提供／供／讓
  - 點一下／點 → 點選／按一下／按
  - 搞手 → 主辦人
  - 波友 → 球友（或「其他用戶」視語境）
  - 開波 → 開始比賽／開始活動
  - 夾隊 → 組隊／組成隊伍

### 標點與格式

- 中文提示語及錯誤訊息優先使用全形標點（例如「：」「（）」）。
- 若句子含英文縮寫或數字，可保留半形字元但維持中文語序。

## 驗證／驗收標準

1. `translations.zh` 內不再出現主要口語化關鍵字（以 grep 檢查為準，允許出現在專有名詞或不可避免語境者需逐一審核）。
2. 前端可正常渲染，不出現缺字（例如 `t('key')` 找不到）。
3. 主要頁面（登入、探索、主辦公開場、球隊、活動詳情）之中文介面文案均為書面語風格，且稱呼一致使用「你」。
